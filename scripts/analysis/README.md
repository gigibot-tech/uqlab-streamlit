# Analysis Scripts

This directory contains scripts for analyzing experiment results from the UQ classification system.

## Available Scripts

### `analyze_my_run.py`

Analyzes experiment results from a single run directory and creates visualizations of AUROC metrics.

**Usage:**
```bash
# From project root
python scripts/analysis/analyze_my_run.py

# Or make it executable and run directly
chmod +x scripts/analysis/analyze_my_run.py
./scripts/analysis/analyze_my_run.py
```

**What it does:**
1. Loads `summary.json` and `training_data.config.json` from `results/my_run/`
2. Extracts AUROC metrics for all uncertainty signals
3. Creates visualizations:
   - **Bar plot**: Compares AUROC scores across signals and uncertainty types
   - **Heatmap**: Shows signal performance matrix
4. Saves outputs to `results/my_run/analysis/`:
   - `auroc_comparison.png` - grouped bar chart
   - `auroc_heatmap.png` - heatmap visualization
   - `auroc_metrics.csv` - metrics in CSV format

- matplotlib
- numpy

## Adding New Analysis Scripts

When adding new analysis scripts:

1. **Follow naming convention**: `analyze_<what>.py`
2. **Add docstring** with usage examples
3. **Update this README** with script description
4. **Make executable**: `chmod +x scripts/analysis/your_script.py`
5. **Add shebang**: `#!/usr/bin/env python3`

## Data Format

Scripts expect experiment results in this structure:

```
results/
└── my_run/
    ├── summary.json              # Main results (AUROC, config, etc.)
    ├── training_data.config.json # Experiment configuration
    ├── results.pt                # PyTorch results tensor
    ├── per_sample_signals.csv    # Per-sample uncertainty scores
    └── analysis/                 # Generated by analysis scripts
        ├── auroc_comparison.png
        ├── auroc_heatmap.png
        └── auroc_metrics.csv
```

### `summary.json` Structure

```json
{
  "config": { ... },
  "auroc_rows": [
    {
      "signal": "mutual_info",
      "aleatoric_auroc": 0.5405,
      "epistemic_auroc": 0.3632,
      "ood_auroc": 0.6640
    },
    ...
  ],
  "one_vs_rest_auroc": [ ... ],
  "macro_f1": [ ... ]
}
```

## Tips

- **Batch analysis**: To analyze multiple runs, create a loop in your script
- **Custom plots**: Import the plotting functions and customize them
- **Export formats**: Change `plt.savefig()` format parameter for PDF, SVG, etc.
- **Interactive mode**: Remove `output_path` parameter to show plots without saving

## Troubleshooting

**Import errors:**
```bash
# Make sure you're in the project root
cd /path/to/uqlab-streamlit
python scripts/analysis/analyze_my_run.py
```

**Missing results:**
```
FileNotFoundError: results/my_run/summary.json
```
→ Check that the run directory exists and contains the required JSON files

**Type warnings:**
The linter may show type warnings for pandas operations - these are safe to ignore as the code works correctly at runtime.
